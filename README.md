# Pansport E-commerce website:

- Ovo je projekat e-commerce sajta za prodaju sportskih suplemenata.
- Sajt omogućava korisnicima da pregledaju proizvode, informišu se o suplementima i kreću kroz stranice kao što su main-page, proizvod, checkout i login.

## Funkcionalnosti:
- Prikaz proizvoda
- Login stranica
- Navigacija između stranica
- Responsive dizajn (prilagođen različitim uređajima)

## Tehnologije:
- HTML
- CSS
- JavaScript
- Bootstrap

## Struktura projekta:
- `main-page.html` – početna stranica
- `login.html` – login stranica
- `proizvod.html` – stranica proizvoda
- `checkout.html` – stranica za kupovinu
- `style.css` – stilovi
- `script.js` – funkcionalnosti
